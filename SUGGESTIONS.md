# Palworld Companion App - Suggestions & Research

Based on a review of popular competitor sites (e.g., PalworldTrainer, Game8, Palworld.gg), the following suggestions are proposed to enhance the application's content and user experience.

## Content Suggestions

### 1. comprehensive Paldeck Database
**Current State:** Sparse data (7 entries).
**Suggestion:** Implement a full database of all 137+ Pals.
**Key Data Points:**
- **Drops:** Essential for farming guides (e.g., High Quality Pal Oil).
- **Partner Skills:** detailed descriptions and effects.
- **Food Requirement:** Hunger meter (1-10).
- **Spawn Locations:** Map integration or text description.
- **Active Skills:** List of learnable moves by level.

### 2. Item Database & Recipes
**Current State:** Missing.
**Suggestion:** A searchable database for items, structures, and equipment.
**Features:**
- **Recipe Trees:** "What do I need to craft X?"
- **Drop Sources:** "Which Pal drops X?"
- **Usage:** "What is X used for?"

### 3. Advanced Breeding Features
**Current State:** Basic calculator exists.
**Suggestion:** Add "Pathfinding" and "Matrix" tools.
**Features:**
- **Shortest Path:** "How do I get Anubis from what I have?" (User inputs their current pals).
- **Matrix View:** Visual grid of breeding combinations.
- **Passive Inheritance:** Calculator for probability of passing down 4 passives.

### 4. Interactive Tiled Map
**Current State:** Basic SVG image.
**Suggestion:** Implement a tiled map (using Leaflet or Mapbox) with high-resolution game map tiles.
**Features:**
- **Zoom/Pan:** Google Maps-style interaction.
- **Filters:** Fast Travel, Dungeons, Skill Fruit Trees, Effigies, Alpha Pals.
- **User Progression:** "Mark as found" for collectibles (requires local storage).

### 5. Guides & Articles
**Current State:** Minimal.
**Suggestion:** Expand guides section.
**Topics:**
- **Base Building:** Best locations (linked to map), layout optimization.
- **Farming:** Ore, Coal, Sulfur farming routes.
- **Combat:** Element type charts, best pals for specific bosses.

## Graphics & UI Suggestions

### 1. Consistent Iconography
- **Element Badges:** Use distinct, colorful icons for Fire, Water, Grass, etc., consistent across the app.
- **Work Suitability:** Use recognizable icons for Kindling, Mining, etc., with level indicators (Lv 1-4).

### 2. Visual Breeding Tree
- **Family Tree Visualization:** A recursive tree view for breeding paths (Grandparents -> Parents -> Child) is highly requested.

### 3. Dark/Light Mode Polish
- Ensure all text, especially in tables and cards, has sufficient contrast in both modes (e.g., use `text-slate-200` on dark backgrounds).

### 4. Hero Images
- Use high-quality in-game screenshots for guide headers and Pal detail pages to improve visual appeal.
