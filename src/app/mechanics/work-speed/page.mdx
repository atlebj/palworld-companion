# Work Speed Explained

Work Speed affects how quickly a Pal completes **work tasks** at your base (crafting, mining, farming, etc.). It does **not** directly increase combat damage or movement speed.

## What Work Speed affects
- Task completion time at workstations
- Gathering speed for certain base tasks
- Overall base throughput (items/hour)

## What Work Speed does NOT affect
- Capture chance
- Combat damage (unless a specific passive states otherwise)
- Player crafting speed (that’s separate)

## Practical optimization
1. Use high-work-suitability Pals for the right stations  
2. Stack work-related passives on dedicated workers  
3. Reduce idle time: shorten travel paths and avoid overcrowding

## Common mistakes
- Using “generalist” Pals with many skills that constantly switch tasks
- Overbuilding stations without enough workers
- Ignoring pathing and storage placement
