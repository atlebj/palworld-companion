# Breeding Trait Inheritance Explained

Breeding inheritance determines which **passive skills** (traits) a baby Pal can receive from its parents. The goal is usually to **combine the best passives** into one optimized Pal.

## What can be inherited
- Passive skills from either parent
- Some outcomes depend on how many passives exist on each parent

## Reliable strategy
- Start with parents that each have **only the passives you want**
- Avoid “junk passives” (they can take a slot)
- Breed in steps: 2 good → 3 good → 4 good

## Common mistakes
- Using parents with too many random passives
- Trying to jump straight to a perfect 4-passive Pal
- Not tracking which parent lines are “clean”
